# Template

## Início rápido

```shell
# instalar dependências
npm install
# ou
yarn install
```

```shell
# iniciar em mode de desenvolvimento
npm run dev
# ou
yarn dev
```

Acessar no browser: **http://localhost:3000/**

### Build

O build sera gerado na pasta: **/build**:

```shell
# Para gerar o build
npm run build
# or
yarn build
```
